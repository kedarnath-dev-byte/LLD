# java-lld-design-patterns

A Java practice repository for object-oriented programming, design patterns, multithreading, and low-level design case studies. The repository contains small focused examples along with larger design exercises such as Parking Lot and Tic Tac Toe.

## Portfolio Role

This repository is maintained by Mamani Kedarnath as a Java low-level design portfolio project focused on OOP, design patterns, multithreading, and machine-coding case studies.

## Topics Covered

- Object-oriented programming
- Access modifiers and constructors
- Inheritance and polymorphism
- Interfaces and abstract classes
- Upcasting and downcasting
- Java generics
- Lambda expressions and streams
- Multithreading basics
- Semaphores
- Design patterns
- Low-level design case studies

## Design Patterns

The repository includes implementations and examples for:

- Adapter pattern
- Decorator pattern
- Factory pattern
- Prototype pattern
- Strategy pattern

## Case Studies

### Parking Lot

Low-level design for a parking lot system with:

- Parking lots, floors, gates, spots, vehicles, tickets, bills, and payments
- Controller, service, repository, model, exception, and strategy layers
- Spot assignment strategy abstraction
- Bill calculation strategy abstraction
- Custom exceptions for missing entities and unavailable spots

### Tic Tac Toe

Low-level design for a Tic Tac Toe game with:

- Board, cell, move, player, bot, and game models
- Game controller and service layers
- Winner checking strategy
- Bot playing strategy abstraction
- Difficulty-level based bot strategy selection
